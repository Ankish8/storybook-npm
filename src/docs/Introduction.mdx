## Why myOperator UI?

### Copy & Paste Components
Unlike traditional component libraries, you own the code. Components are copied directly into your project, giving you full control to customize.

### Built on Best Practices
- **Radix UI** - Accessible, unstyled primitives
- **Tailwind CSS** - Utility-first styling
- **CVA** - Type-safe component variants
- **TypeScript** - Full type definitions

### Bootstrap Compatible
Seamlessly integrate with existing Bootstrap projects. Our CLI detects Bootstrap and configures Tailwind to avoid conflicts.

## Quick Install

```bash
# Initialize your project
npx myoperator-ui init

# Add all components at once
npx myoperator-ui add --all

# Or add specific components
npx myoperator-ui add button badge table
```
