# myOperator UI

A collection of production-ready React components built with **Tailwind CSS**, **Radix UI**, and **TypeScript**.

```bash
npx myoperator-ui init
npx myoperator-ui add button badge table
```

**[Get Started →](?path=/docs/getting-started--docs)** · [CLI Reference](?path=/docs/cli-reference--docs) · [MCP Server](?path=/docs/mcp-server--docs)

---

## Why myOperator UI?

### Own Your Code
Components are copied directly into your project. No dependency lock-in — full control to customize everything.

### Accessible by Default
Built on Radix UI primitives with proper ARIA attributes, keyboard navigation, and screen reader support.

### Bootstrap Compatible
Works alongside existing Bootstrap projects. The CLI auto-configures Tailwind to avoid style conflicts.

### TypeScript First
Full type definitions with autocomplete and IntelliSense. Catch errors at build time, not runtime.

---

## Quick Install

Initialize your project and add components in seconds:

```bash
# Initialize (installs dependencies, configures Tailwind)
npx myoperator-ui init

# Add specific components
npx myoperator-ui add button badge

# Or add ALL components at once
npx myoperator-ui add --all
```

---

## Usage

```tsx
import { Button } from '@/components/ui/button'
import { Badge } from '@/components/ui/badge'

export default function App() {
  return (
    <div>
      <Button variant="default">Click me</Button>
      <Badge variant="active">Active</Badge>
    </div>
  )
}
```
