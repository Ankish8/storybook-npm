## Why myOperator UI?

### Copy & Paste Components
Unlike traditional component libraries, you own the code. Components are copied directly into your project, giving you full control to customize.

### Built on Best Practices
- **Radix UI** - Accessible, unstyled primitives
- **Tailwind CSS** - Utility-first styling
- **CVA** - Type-safe component variants
- **TypeScript** - Full type definitions

### Bootstrap Compatible
Seamlessly integrate with existing Bootstrap projects. Our CLI detects Bootstrap and configures Tailwind to avoid conflicts.

## Quick Install

```bash
# Initialize your project
npx myoperator-ui init

# Add all components at once
npx myoperator-ui add --all

# Or add specific components
npx myoperator-ui add button badge table
```

## Components

We offer 13 production-ready components:

| Component | Description |
|-----------|-------------|
| **Badge** | Status indicators with active, failed, and disabled variants |
| **Button** | Customizable button with variants, sizes, icons, and loading states |
| **Checkbox** | Tri-state checkbox with label support |
| **Collapsible** | Expandable/collapsible sections |
| **Dropdown Menu** | Actions and options menu |
| **Input** | Basic input component |
| **Multi-Select** | Multi-select with search and tags |
| **Select** | Single select dropdown |
| **Select Field** | Select with label and validation |
| **Table** | Composable table with sorting and sticky columns |
| **Tag** | Event labels with optional bold prefix |
| **Text Field** | Input with label, icons, and validation |
| **Toggle** | Switch component for boolean inputs |
