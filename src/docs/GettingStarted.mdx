# Getting Started

myOperator UI is a collection of reusable React components built with Tailwind CSS. Install components directly into your project with our CLI tool.

## Quick Start

### 1. Initialize your project

```bash
npx myoperator-ui init
```

This will:
- Install required dependencies (tailwindcss, class-variance-authority, clsx, etc.)
- Create a `tailwind.config.js` if needed
- Set up CSS variables for theming
- Create a `components/ui` folder

### 2. Add components

```bash
# Add a specific component
npx myoperator-ui add button

# Add multiple components
npx myoperator-ui add button badge table

# Or add ALL components at once
npx myoperator-ui add --all
```

> **Tip:** See the **[CLI Reference →](?path=/docs/cli-reference--docs)** for all available options.

### 3. Import and use

```tsx
import { Button } from '@/components/ui/button'

export default function App() {
  return (
    <Button variant="default" size="default">
      Click me
    </Button>
  )
}
```

## Manual Installation

If you prefer to install manually:

### 1. Install dependencies

```bash
npm install tailwindcss class-variance-authority clsx tailwind-merge
npm install @radix-ui/react-slot
```

### 2. Configure Tailwind

Add the following to your `tailwind.config.js`:

```js
module.exports = {
  content: ['./src/**/*.{js,ts,jsx,tsx}'],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 3. Copy components

Copy the component files from the Storybook docs into your `components/ui` folder.

## TypeScript

All components are written in TypeScript with full type definitions. Props are exported for each component:

```tsx
import { Button, type ButtonProps } from '@/components/ui/button'
```

---

## Customization

### Variants

Components use `class-variance-authority` for variants:

```tsx
<Button variant="outline" size="lg">
  Large Outline Button
</Button>
```

### Styling

Override styles with className:

```tsx
<Button className="bg-purple-500 hover:bg-purple-600">
  Custom Purple
</Button>
```
