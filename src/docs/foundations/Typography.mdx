# Typography

Like in other UI systems, typography works by the principle of accessibility before aesthetics. Therefore, the text should be readable and help the user understand what's important by well-contrasted size and color hierarchy.

---

## Font Family

We use **Source Sans Pro** for our UI hierarchy. Source Sans Pro is a sans-serif typeface created by Paul D. Hunt for Adobe Systems. It provides excellent readability across all sizes.

## Usage in Code

### CSS Classes

Apply typography styles using the predefined CSS classes:

```css
/* Typography classes are available globally */
.text-headline-small { /* 24px / 32px, SemiBold */ }
.text-headline-medium { /* 28px / 36px, SemiBold */ }
.text-headline-large { /* 32px / 40px, SemiBold */ }

.text-body-small { /* 12px, Regular */ }
.text-body-medium { /* 14px, Regular */ }
.text-body-large { /* 16px, Regular */ }

.text-title-small { /* 14px, SemiBold */ }
.text-title-medium { /* 16px, SemiBold */ }
.text-title-large { /* 18px, SemiBold */ }

.text-label-small { /* 10px, SemiBold */ }
.text-label-medium { /* 12px, SemiBold */ }
.text-label-large { /* 14px / 20px, SemiBold */ }

.text-display-small { /* 36px / 44px, Regular */ }
.text-display-medium { /* 45px / 52px, Regular */ }
.text-display-large { /* 57px / 64px, Regular */ }
```

### TypeScript

Import typography tokens from the typography module:

```typescript
import { typography, headline, body, title, label, display } from '@/lib/typography'

// Access specific styles
const headlineSmall = headline.small
// { fontFamily, fontSize: '24px', lineHeight: '32px', fontWeight: 600, cssClass: 'text-headline-small' }

// Access font weights
const regular = typography.fontWeight.regular  // 400
const semibold = typography.fontWeight.semibold  // 600
```

### Tailwind Classes

Components use Tailwind utility classes that inherit the global Source Sans Pro font:

```tsx
// Tailwind classes work with Source Sans Pro
<p className="text-sm font-normal">Body text (14px, Regular)</p>
<h2 className="text-2xl font-semibold">Heading (24px, SemiBold)</h2>
<label className="text-xs font-semibold">Label (12px, SemiBold)</label>
```

## Usage Guidelines

<div style={{ backgroundColor: '#F6FEF9', border: '1px solid #75E0A7', borderRadius: '8px', padding: '20px', marginTop: '16px' }}>
  <div style={{ fontWeight: 600, fontSize: '16px', color: '#067647', marginBottom: '12px' }}>Best Practices</div>
  <ul style={{ margin: 0, paddingLeft: '20px', color: '#181D27', fontSize: '14px', lineHeight: '1.6' }}>
    <li>Avoid using text size smaller than 14px (except for labels and captions)</li>
    <li>Don't underline words. For typographic emphasis, use <strong>bold text</strong></li>
    <li>Don't use 2 different sizes of text in the same line</li>
    <li>Use semantic text colors for consistent hierarchy</li>
    <li>Maintain proper contrast ratios for accessibility (WCAG 2.1 AA)</li>
  </ul>
</div>

<div style={{ backgroundColor: '#FEF3F2', border: '1px solid #FDA29B', borderRadius: '8px', padding: '20px', marginTop: '16px' }}>
  <div style={{ fontWeight: 600, fontSize: '16px', color: '#B42318', marginBottom: '12px' }}>Avoid</div>
  <ul style={{ margin: 0, paddingLeft: '20px', color: '#181D27', fontSize: '14px', lineHeight: '1.6' }}>
    <li>Mixing font families - always use Source Sans Pro</li>
    <li>Using custom font weights outside of 400 and 600</li>
    <li>Applying underline for emphasis (reserve for links)</li>
    <li>Using very light or very bold weights</li>
  </ul>
</div>
